### ğŸ”§ Code Contributions

#### Getting Started

1. **Fork the repository**
2. **Clone your fork:**
   ```bash
   git clone https://github.com/YOUR-USERNAME/reversesalestaxcalc.git
   ```
3. **Create a branch:**
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes**
5. **Test thoroughly** by opening index.html in multiple browsers
6. **Commit your changes:**
   ```bash
   git commit -m "Add: brief description of changes"
   ```
7. **Push to your fork:**
   ```bash
   git push origin feature/your-feature-name
   ```
8. **Create a Pull Request**

#### Code Style Guidelines

- Use **semantic HTML5** elements
- Follow **existing CSS patterns** (BEM-like naming)
- Write **clean, commented JavaScript**
- Ensure **mobile responsiveness**
- Maintain **accessibility** (ARIA labels, semantic markup)
- Test in **multiple browsers** (Chrome, Firefox, Safari, Edge)

### Testing Checklist

Before submitting a PR, ensure:

- [ ] Calculator produces accurate results
- [ ] Tested with various tax rates (0%, 5%, 10%, 25%)
- [ ] Tested with edge cases (negative numbers, very large numbers)
- [ ] Mobile responsive (test on actual device or DevTools)
- [ ] Works in Chrome, Firefox, Safari, Edge
- [ ] No console errors
- [ ] Accessibility checked (keyboard navigation, screen readers)
- [ ] SEO meta tags intact
- [ ] Links to [reversesalestaxcalc.com](https://reversesalestaxcalc.com/) work correctly

### Tax Rate Updates

If updating tax rates:

- Cite official sources (government websites)
- Include effective date
- Update both display tables and any embedded rates
- Test calculations with new rates
